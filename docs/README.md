## Project Overview

This is a Laravel 12 invoice management application with:
- 14 Filament v4 admin resources
- 4 dashboard widgets
- 5 custom form components
- Comprehensive Blade component system
- Full Peppol (Pan-European Public Procurement OnLine) support

For the main project README, see [../README.md](../README.md).
